Matrícula Web Crawler
=====================

Implementação de função que vasculham as páginas do [Matrícula Web][mweb] coletando informações sobre disciplinas.

[mweb]: https://matriculaweb.unb.br


## Hierarquia das classes
- UnB
  - Campus
    - Departamento
      - Disciplina
  - Curso
    - Habilitação
      - Currículo
        - Disciplina
